# E-commerce-SyntaxWear

Este é um projeto simples de um e-commerce desenvolvido a partir de um design no Figma. O objetivo é replicar o layout e a experiência visual proposta pelo design, utilizando HTML e CSS puros, com foco em responsividade e organização modular de estilos.

## 📁 Estrutura do Projeto

- `index.html` - página principal do site.
- `css/` - pasta com todos os estilos:
  - `reset.css` - estilização base para normalização de elementos.
  - `variables.css` - variáveis CSS (imprime fonte e cores globais).
  - `base.css` - estilos globais reutilizáveis (botões, tipografia, etc.).
  - `components/` - estilos para componentes específicos como header, hero, produtos, footer, etc.
- `images/` - imagens utilizadas no layout, incluindo banners, produtos e ícones.

## 🚀 Tecnologias Utilizadas

- **HTML5**
- **CSS3** (Grid, Flexbox, variáveis, media queries)
- **Fontes Google Fonts** (Ubuntu)

## 🖥️ Funcionalidades e Características

- Layout responsivo adaptado para diversas larguras de tela.
- Navegação principal com versão para mobile (menu hambúrguer).
- Seções de destaque como hero, categorias e grid de produtos.
- Footer informativo com assinaturas de e-mail e links úteis.

## 📐 Referência de Design

## 🛠️ Como Usar

1. Clone este repositório.
2. Abra `index.html` no navegador de sua escolha.
3. Ajuste ou complemente estilos conforme necessário.
